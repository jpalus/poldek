What is poldek?

poldek is a full-featured frontend to RPM (including RPM5) package manager. Its core features are similar to every modern package updater. Features include package installation, upgrading and removal with automatic dependency handling, querying/ search the package database and many more.

poldek is a primary package tool for PLD Linux Distribution.

Last poldek release is POLDEK_VERSION (released POLDEK_DATE). See the NEWS for a list of major changes in this release.
